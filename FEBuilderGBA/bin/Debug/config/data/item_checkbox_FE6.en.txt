//特性1
01=Equippable	Register as a weapon item that can be equipped.\r\nEnable to equip items.
02=Attacks res	Register as an item to perform a magic attack.\r\nIf enabled, Str-Res will be Damage\r\nIf disabled, Str-Def will be Damage.
03=Staff	Sets item as a staff.\r\nComplex settings are required to create the effect of using a staff.\r\n(Do not change)
04=Unbreakable	Gives an item infinite uses.
05=Unsellable	Prevents an item from being sold.
06=Brave Effect	Attacks are followed up with a second attack.\r\nBy default, this is used by Brave Weapons.
07=Magic Damage	Adjusts damage formula to:\r\nPower-Res:Damage\r\nNo weapon exp will be granted.\r\nBy default, this is used for magic swords like the Light Brand.
08=Uncounterable	Prevents foe from counterattacking.\r\nBy default, this is used by long-range weapons like Ballistae and Bolting.

11=Reaver Effect	Reverses the Weapon triangle.\r\nBy default, this is used by the Reaver weapons.
12=Disable Hammerne	Disables Hammerne staff from being used on item.
13=Manakete Lock	Restricts item use to only those with the Manakete Lock set.\r\nBy default, this is set for the Firestone, Magestone, and Divinestone.
14=Roy Lock	Restricts item use to only those with the Roy Lock set.\r\nBy default, this is set for Rapier and Binding Blade.
15=Myrmidon/Swordmaster Lock	Restricts item use to only those with the Myrmidon/Swordmaster lock set.\r\nBy default, this is set for Wo Dao.
16=Zephiel Lock	Restricts item use to only those with the Zephiel Lock set.\r\nBy default, this is set for Exaccus.
17=Negate Flying Effectiveness	When equipped, negates Bow effectiveness against Flying units.\r\nBy default, this is set for the Delphi Shield.
18=Negate Criticals	When equipped, prevents enemy critical hits.\r\nBy default, this is set for the Iron Rune.
