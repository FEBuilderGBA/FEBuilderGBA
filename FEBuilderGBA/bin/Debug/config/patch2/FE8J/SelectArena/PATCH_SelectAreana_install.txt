NAME=闘技場の対戦相手選出ルーチンの改善(Install)
NAME.en=Improved opponent selection routines in the arena(Install)
NAME.zh=Improved opponent selection routines in the arena(Install)

INFO=このパッチは闘技場の敵クラスとのマッチルーチンを変更します。\r\n1. 闘技場の敵をバニラの様にリストから読み込むのではなく、クラステーブル全体から選出するようにします。\r\n2. あなたが上級職であっても、下級職と戦うことができます。\r\n3. あなたが下級職だった場合、下級職としか戦えません。\r\n4. 武器レベルがなく、武器を持てないクラスは選出リストから除外されます。そのため、市民などのクラスが選出されることはありません。\r\n4. 特徴のビットフラグを設定することで、超闘技場(Super Arena)のみに登場するレアクラスを設定できます。ディフォルトはエイリークの武器ロックのビットマスクになっているので、あなたの作品に合わせて変更してください。\r\n5. 特徴のビットフラグを設定することで、特定グローバルフラグが設定されているときにのみ戦えるスーパーレアクラスを設定できます。スーパーレアは、エンディング後や特定の条件後に戦えるクラスを定義するのに使います。たとえば、魔王が闘技場に出てきたら困る場合などです。ディフォルトは、フラグ0x73で、エフラムの武器ロックに設定されているので、こちらもあなたの作品に合わせて変更してください。\r\n6. 闘技場に選出しないという方法はありません。どうしてもそれを設定したい場合、武器レベルを取り上げるか、スーパーレアクラスの機能を使わないことで表現してください。つまり、そのフラグを永久に有効にしなければ、永久に選出されないことになります。\r\nこのパッチを使うには、「闘技場の敵の武器選出の改善」パッチが必要です。\r\nもし、インストールを忘れると、反撃できない敵が選出されてしまうでしょう。\r\n選出される下級職を強くしたい場合は、「Arenaで強い敵ユニットを出現させる」パッチを利用して、class capsを外してください。\r\nステータス上限を超えて成長できるので、下級職でも敵がとても強くなります。\r\nこのパッチは、「闘技場で特殊なレア敵を出現させる」を無効化します。なぜなら、その上位版だからです。\r\nレア敵を下級職の上書きで行っていたが、数が増えすぎて置き換える下級職が足りなくなったので、このパッチを開発しました。

INFO.en=This patch changes the arena enemy class match routine.\r\n1. Arena enemies are picked from the entire class table instead of loading from a list like vanilla.\r\n2. Even if you are a Promoted class, you can fight a un-Promoted class.\r\n3. If you were a un-Promoted class, you can only fight with un-Promoted classes.\r\n4. Classes with no weapon level and no weapons will be removed from the selection list.\r\nSo classes such as Citizens are never elected.\r\n4. By setting the feature bit flag, you can set a rare class that appears only in Super Arena.\r\nThe default is Erika's weapon lock bitmask, so change it to suit your work.\r\n5. By setting the trait bit flag, you can set a super rare class that can only be fought when a specific global flag is set.\r\nSuper rares are used to define classes that can be fought after endings or after certain conditions.\r\nFor example, if you are in trouble if the Demon King appears in the arena.\r\nThe default is flag 0x73, which is set to Ephraim's Weapon Lock, so change that to suit your creations as well.\r\n6. There is no way not to be elected to the arena.\r\nIf you really want to set it, express it by removing the weapon level or not using the Super Rare class feature.\r\nIn other words, if you don't permanently enable that flag, you're permanently not elected.\r\nThis patch requires the "Improvements in selection of weapons for enemies in the arena" patch.\r\nIf you forget to install it, enemies who cannot counterattack will be selected.\r\nIf you want to strengthen the elected un-promoted class, please use the "Make strong enemy units appear in Arena" patch and remove the class caps.\r\nSince you can grow beyond the status limit, enemies will become very strong even in un-promoted class.\r\nThis patch disables "Spawn special rare enemy in Arena".\r\nBecause it's the higher version.\r\nI used to overwrite the lower classes for rare enemies, but the number of them increased too much and I didn't have enough junior classes to replace them, so I developed this patch.
