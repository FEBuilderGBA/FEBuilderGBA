INFO.en=Edits the table created by "Level Cap Editor Installer" which allows for variable level caps per unit and/or class.\r\nIf the unit is a trainee, their level cap won't be adjusted.\r\nTo set a level cap per class, set Unit byte to 0xFF.\r\nTo set a level cap per unit, set Class byte to 0xFF.\r\nLevel cap can also be set by specifying both unit and class for by-character-class level cap restrictions.\r\nAny unit/class that isn't added to this table will use the default level cap (20 if unpatched)\r\nNote: Do not set ANY for both units and classes.\r\n

INFO.zh=将等级上限变更为职业和人物。\r\n规则\r\n如果有见习标志，则该检查将被忽略。\r\n在人物和职业上设置最大级别。\r\n如果要为每个职业设置级别，请将0xFF（ANY）设置为该人物。\r\n如果你想为每个人物设置电平，请为该类设置0xFF（ANY）\r\n对于那些与此表不匹配的，将使用默认级别限制（通常是20。它可以通过修补程序进行更改）\r\nNote: Do not set ANY for both units and classes.\r\n
