NAME=名前入力画面の最大値を変更する
INFO=イベント「名前入力画面の呼び出し」で、入力できる名前を、5文字から9文字に変更します。

//アドレスパッチとして実装します
TYPE=SWITCH
TAG=#ENGINE

COMBO=Default|OFF|8文字|ON8|9文字|ON9

DETERMINATION_ADDRESS=0x044800

//max name length
OFF:0x44800=0x5
ON8:0x44800=0x8
ON9:0x44800=0x9

//max cursor position
OFF:0x482bA=0x5
ON8:0x482bA=0x8
ON9:0x482bA=0x9
