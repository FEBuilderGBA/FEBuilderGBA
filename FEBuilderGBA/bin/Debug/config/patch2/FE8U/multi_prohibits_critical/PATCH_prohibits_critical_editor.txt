NAME=必殺を禁止するクラスを複数定義する 20200719(Editor)
NAME.en=Define multiple classes that prohibit critical 20200719(Editor)
NAME.zh=Define multiple classes that prohibit critical 20200719(Editor)

INFO=指定したクラスまたはユニットへの必殺を無効にします。
INFO.en=Disables critical for the specified class or unit.
INFO.zh=Disables critical for the specified class or unit.

TYPE=STRUCT
IF:0x2ACAC=0x00 0x4B 0x18 0x47

//データへのポインタ
POINTER=$FGREP4END prohibits_critical.dmp

//データサイズ(10進数)
DATASIZE=2

//データ個数 00 が出てくるまで
DATACOUNT=$GREP2 0xff 0x00

//リストに表示されるのはCLASSです
LISTNAME=$W0:W_UNITCLASS

//データを定義します
B0:UNIT=UNIT(00=ANY)
B1:CLASS=CLASS(00=ANY)

AUTHOR=7743
